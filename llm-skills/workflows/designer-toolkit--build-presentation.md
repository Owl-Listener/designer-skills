# build-presentation
Structure a design presentation.
## Steps
1. **Audience analysis** — Identify what the audience cares about and knows.
2. **Story arc** — Define the narrative structure using the presentation-deck skill (see skills/designer-toolkit--presentation-deck.md).
3. **Key messages** — Distill to 3-5 main takeaways.
4. **Slide outline** — Plan each slide's purpose and content using the presentation-deck skill (see skills/designer-toolkit--presentation-deck.md).
5. **Rationale slides** — Include decision rationale using the design-rationale skill (see skills/designer-toolkit--design-rationale.md).
6. **Copy review** — Refine slide text using the ux-writing skill (see skills/designer-toolkit--ux-writing.md).
## Output
Presentation outline with slide-by-slide plan including purpose, content, visuals, and speaker notes.
Consider following up with the write-rationale workflow to deepen the reasoning sections.
