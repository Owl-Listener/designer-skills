# prototype-plan
Create a prototyping and testing plan.
## Steps
1. **Strategy** — Choose fidelity and method using the prototype-strategy skill (see skills/prototyping-testing--prototype-strategy.md).
2. **Flows** — Map user flows to prototype using the user-flow-diagram skill (see skills/prototyping-testing--user-flow-diagram.md).
3. **Wireframes** — Specify wireframe layouts using the wireframe-spec skill (see skills/prototyping-testing--wireframe-spec.md).
4. **Test scenarios** — Write usability tasks using the test-scenario skill (see skills/prototyping-testing--test-scenario.md).
5. **Accessibility** — Plan accessibility testing using the accessibility-test-plan skill (see skills/prototyping-testing--accessibility-test-plan.md).
6. **Timeline** — Create a prototyping and testing schedule.
## Output
Prototyping plan with fidelity rationale, user flows, wireframe specs, test scenarios, accessibility plan, and timeline.
Consider following up with the test-plan workflow for detailed usability testing or the evaluate workflow for expert review.
