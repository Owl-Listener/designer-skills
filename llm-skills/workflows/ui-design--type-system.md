# type-system
Create a complete typography system.
## Steps
1. **Scale** — Build a modular type scale using the typography-scale skill (see skills/ui-design--typography-scale.md).
2. **Hierarchy** — Define hierarchy levels using the visual-hierarchy skill (see skills/ui-design--visual-hierarchy.md).
3. **Responsive** — Adapt scale across breakpoints using the responsive-design skill (see skills/ui-design--responsive-design.md).
4. **Spacing** — Define text spacing relationships using the spacing-system skill (see skills/ui-design--spacing-system.md).
5. **Grid alignment** — Align to baseline grid using the layout-grid skill (see skills/ui-design--layout-grid.md).
6. **Document** — Output complete type system with usage rules.
## Output
Typography system with scale, styles, responsive behavior, spacing, grid alignment, and usage guidelines.
Consider following up with the design-screen workflow to apply the type system.
