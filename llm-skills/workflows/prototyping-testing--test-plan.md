# test-plan
Design a complete usability testing plan.
## Steps
1. **Objectives** — Define what you need to learn.
2. **Method** — Choose testing approach using the prototype-strategy skill (see skills/prototyping-testing--prototype-strategy.md).
3. **Scenarios** — Write test tasks using the test-scenario skill (see skills/prototyping-testing--test-scenario.md).
4. **Click tests** — Design findability tests using the click-test-plan skill (see skills/prototyping-testing--click-test-plan.md).
5. **Accessibility** — Include accessibility testing using the accessibility-test-plan skill (see skills/prototyping-testing--accessibility-test-plan.md).
6. **Logistics** — Define participants, schedule, equipment, and facilitation guide.
## Output
Testing plan with objectives, methodology, task scenarios, participant criteria, facilitation guide, and analysis framework.
Consider following up with the evaluate workflow for expert review before user testing.
