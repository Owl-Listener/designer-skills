# write-case-study
Create a portfolio case study.
## Steps
1. **Outline** — Structure the case study narrative using the case-study skill (see skills/designer-toolkit--case-study.md).
2. **Challenge** — Define the problem and context clearly.
3. **Process** — Highlight key moments of insight and decision using the design-rationale skill (see skills/designer-toolkit--design-rationale.md).
4. **Solution** — Describe the final design and its key features.
5. **Impact** — Quantify and qualify the results.
6. **Polish copy** — Refine the writing using the ux-writing skill (see skills/designer-toolkit--ux-writing.md).
## Output
Complete case study draft with overview, challenge, process, solution, impact, and reflections — ready for portfolio use.
Consider following up with the build-presentation workflow to create a presentation version.
