# design-screen
Design a complete screen layout from a description.
## Steps
1. **Structure** — Define layout grid using the layout-grid skill (see skills/ui-design--layout-grid.md).
2. **Hierarchy** — Establish visual priority using the visual-hierarchy skill (see skills/ui-design--visual-hierarchy.md).
3. **Typography** — Apply type styles using the typography-scale skill (see skills/ui-design--typography-scale.md).
4. **Color** — Apply color system using the color-system skill (see skills/ui-design--color-system.md).
5. **Spacing** — Apply consistent spacing using the spacing-system skill (see skills/ui-design--spacing-system.md).
6. **Responsive** — Define behavior across breakpoints using the responsive-design skill (see skills/ui-design--responsive-design.md).
7. **Dark mode** — Specify dark mode adaptation using the dark-mode-design skill (see skills/ui-design--dark-mode-design.md).
## Output
Complete screen specification with layout, hierarchy, typography, color, spacing, responsive behavior, and dark mode.
Consider following up with the responsive-audit workflow to verify the design.
