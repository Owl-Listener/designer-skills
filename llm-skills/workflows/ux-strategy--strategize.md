# strategize
Develop a comprehensive UX strategy.
## Steps
1. **Vision** — Articulate the north-star using the north-star-vision skill (see skills/ux-strategy--north-star-vision.md).
2. **Landscape** — Analyze competitors using the competitive-analysis skill (see skills/ux-strategy--competitive-analysis.md).
3. **Map** — Create experience map using the experience-map skill (see skills/ux-strategy--experience-map.md).
4. **Principles** — Define guiding principles using the design-principles skill (see skills/ux-strategy--design-principles.md).
5. **Opportunities** — Evaluate and prioritize using the opportunity-framework skill (see skills/ux-strategy--opportunity-framework.md).
6. **Metrics** — Define success measures using the metrics-definition skill (see skills/ux-strategy--metrics-definition.md).
7. **Brief** — Consolidate into a design brief using the design-brief skill (see skills/ux-strategy--design-brief.md).
## Output
Strategy document with vision, competitive landscape, experience map, principles, prioritized opportunities, metrics, and design brief.
Consider following up with the benchmark workflow or the frame-problem workflow.
